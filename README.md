# Actividad 17 — Ta-Te-Tí (2D) y Paddle 3D (p5.js)

Proyecto web que implementa dos juegos pedidos en la actividad: Ta-Te-Tí en 2D y Paddle en 3D con WEBGL, utilizando p5.js para renderizado y lógica de interacción.

## Estructura
- index.html
- styles.css
- script.js

## Cómo ejecutar
1) Abrir `index.html` directamente en el navegador.
2) Opcionalmente usar la extensión **Live Server** de VSCode para recarga automática.

## Uso
- Selector “Juego” para alternar entre **Ta-Te-Tí** y **Paddle 3D**.
- Botón **Reiniciar** para comenzar de nuevo.
- **Ta-Te-Tí:** hacer clic en una casilla para colocar la marca. El estado del turno y el resultado se muestran en pantalla.
- **Paddle 3D:** P1 con teclas **W/S**; P2 con **↑/↓**. El marcador se actualiza al anotar.

## Tecnologías
- p5.js (2D/WEBGL)
- HTML5, CSS3, JavaScript
